SealClubber allows for fast on-the-fly switching of a pally's seal.  Simply hover over the button and use your mousewheel.

SealClubber is intended to be used in a macro, instead of rewriting the macro every time you need to change seals.  Just add @/click SealClubber@ to your macro.
